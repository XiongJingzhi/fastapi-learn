# 阶段 0.1 学习指南

## 📚 学习内容概览

本阶段我们将学习 **并发 vs 并行** 的区别。

**为什么重要？**
- 这是理解 FastAPI 异步编程的基础
- 帮助你选择正确的编程模式（多线程/多进程/异步）
- 理解为什么 FastAPI 能高效处理大量请求

---

**重点关注**：
- 并发的定义
- 并行的定义
- 两者的关键区别

## 💡 学习提示

### 如果觉得理解困难：

**策略 A：换个角度**
- 想想其他生活场景
- 比如开车、收银员、运动员等

**策略 B：画图理解**
- 画出并发的时间线（任务 A - 任务 B - 任务 A...）
- 画出并行的示意图（任务 A 和任务 B 并排）

**策略 C：请求帮助**
- 在团队中提问
- architect 可以提供更多图示
- senior-dev 可以提供代码示例

---

### 如果觉得太简单：

**挑战模式**：
- 研究 Python 的 GIL 是如何工作的
- 思考：为什么 Python 不去掉 GIL？
- 查看 CPython 的源代码实现
- 考虑：Go 语言的 goroutine 是并发还是并行？

---

## ✅ 完成标志

3. ✅ 解释 Python GIL 的影响
4. ✅ 知道什么时候用并发/并行
